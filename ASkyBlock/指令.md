# 指令

| 指令 |   参数   |          效果          |
| :--: | :------: | :--------------------: |
| /sky |          |         主命令         |
| /is  |   help   |        查看帮助        |
| /is  |  create  |      创建一个新岛      |
| /is  |  delete  |     删除你站在的岛     |
| /is  |  expel   |   踢出你岛屿的闯入者   |
| /is  |   home   |        回到岛上        |
| /is  |   info   | 查看你所站在岛屿的信息 |
| /is  |  leave   |        回到主城        |
| /is  | sethome  |   设置你岛屿的出生点   |
| /is  | settings |  设置你岛屿的相关信息  |
